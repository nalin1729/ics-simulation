In this repository, you will find variants of a methodology for simulation and calculation
of spectra generated by inverse Compton Scattering: https://www.cv.nrao.edu/course/astr534/InverseCompton.html.

This work was done in collaboration with Professors Balsa Terzic and Geoffery Krafft, of
Old Dominion University and Jefferson Labs. For more information, check out the readme files in
each of the sub-directories for specifications of what code is stored there. Also check out
the published paper from Physical Review Accelerators and Beams March 2018 release!
